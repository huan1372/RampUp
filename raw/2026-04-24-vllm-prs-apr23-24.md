## Performance & Optimization

### PR #40654 — Avoid seq_lens_cpu GPU→CPU Sync
**Author:** njhill (vLLM member)  
**Category:** Speculative decoding, sync elimination

Removes unnecessary GPU-to-CPU synchronization when computing `seq_lens_cpu` in the speculative decoding pipeline. CPU-GPU synchronization forces a pipeline bubble — the GPU must drain before the CPU can read the value, stalling the next dispatch. This optimization eliminates a blocking sync point, reducing TTFT and TPOT overhead specifically for speculative decode paths. Part of the broader "zero CPU-GPU sync" goal in MRV2.

### PR #40574 — Move CUTLASS MoE to fused_moe/experts/

## Metrics & Observability

### PR #40662 — Unified Synthetic Acceptance Rate for V1 and V2
**Author:** benchislett  
**Category:** Speculative decoding metrics

Unifies the synthetic acceptance rate metric across V1 (original engine) and V2 (MRV2) speculative decoding backends. Previously, acceptance rate was computed differently in each backend, making cross-backend comparisons unreliable. This enables consistent measurement of speculative decoding efficiency across engine versions.

## Infrastructure

### PR #40730 — Remove asyncio Infrastructure from Async EPLB
**Author:** SageMoore  
**Category:** EPLB, async infrastructure

Simplifies the Async Expert Parallel Load Balancing (EPLB) implementation by removing unnecessary asyncio event loop infrastructure. EPLB dynamically redistributes expert weights across GPUs during serving based on load signals. The asyncio overhead was adding scheduling latency without benefit; direct synchronous calls (or native CUDA async) replace the asyncio layer.

## Bug Fixes

### PR #40664 — Fix Qwen3 MoE Gate Called Twice
**Author:** jikunshang  
**Category:** Bug fix, MoE

Corrects a duplicate gate computation in Qwen3's MoE routing. The gate (router network that decides which experts each token goes to) was being called twice per forward pass, doubling router FLOP cost. Functional fix: no output change, but ~2× reduction in router compute overhead for Qwen3 MoE models.
